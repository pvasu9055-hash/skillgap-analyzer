# Skill Gap Analyzer

A full-stack application that analyzes the gap between a user's current skills and job required skills.

## Features
- Compare user skills with job skills
- Shows match percentage
- Identifies missing skills
- Suggests learning resources

## Tech Stack
Frontend:
- React
- JavaScript
- CSS

Backend:
- Spring Boot
- REST API
- Gradle

Database:
- H2 Database

DevOps:
- Docker
- GitHub

## How It Works
1. User enters current skills
2. User enters job required skills
3. Application analyzes the gap
4. Shows:
   - Match percentage
   - Matched skills
   - Missing skills
   - Skills to learn
