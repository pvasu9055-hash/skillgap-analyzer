# 🚀 Skill Gap Analyzer

A full-stack application that analyzes the gap between a user's current skills and the skills required for a target job.

---

## 📌 Features

✔ Compare user skills with required job skills  
✔ Identify missing skills  
✔ Suggest improvements  
✔ Clean and simple interface  

---

## 🛠 Tech Stack

- Java
- Gradle
- Docker

---

## 📂 Project Structure

src → Application source code  
gradle → Build tools  
Dockerfile → Container setup  

---

## ▶ How to Run

1. Clone the repository

git clone https://github.com/pvasu9055-hash/skillgap-analyzer

2. Go to the project folder

cd skillgap-analyzer

3. Run using Gradle

./gradlew run
